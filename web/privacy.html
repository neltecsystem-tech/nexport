プライバシーポリシー

最終更新日: 2026年5月15日
事業者名: 株式会社NELTEC（以下「当社」）

当社は、社内向け業務支援アプリケーション「NexPort」（以下「本アプリ」）の提供にあたり、利用者の個人情報を以下のとおり適切に取り扱います。

1. 取得する利用者情報

本アプリは、以下の情報を取得します。

項目取得目的
メールアドレス、氏名、表示名、プロフィール画像アカウント認証、社内コミュニケーション上の本人特定
チャンネルメッセージ、ダイレクトメッセージ、報告書、業務日報、回覧板投稿、コメント社内情報共有、業務管理
勤怠記録（出退勤時刻、休憩、備考）勤怠管理、給与計算の基礎データ
業務スケジュール、タスク、面談・面接記録業務計画・進捗管理
名刺画像および当該画像から自動抽出された氏名・会社名・連絡先等社内名刺共有、取引先連絡管理
プッシュ通知用デバイストークン通知配信
デバイス情報（OS種別、バージョン、機種名）、IPアドレス、アクセスログ不具合解析、不正利用検知、サービス改善

2. 利用目的

  • 当社およびグループ会社の業務に必要な情報共有・コミュニケーション提供
  • 勤怠、業務報告、スケジュール等の管理
  • 利用者本人への業務上の連絡・通知の配信
  • サービスの保守、品質改善、不具合解析
  • 不正利用、規約違反行為の検知・防止
  • 法令に基づく対応

3. 第三者提供および外部委託先

当社は、法令に基づく場合を除き、利用者の同意なく個人情報を第三者に提供しません。ただし、サービス運営上必要な範囲で、以下の外部サービス事業者にデータ処理を委託しています。

委託先所在地委託内容
Supabase Inc.米国データベース、認証、ファイルストレージ、サーバレス関数（Edge Functions）
Google LLC（Firebase Cloud Messaging / Gemini API）米国プッシュ通知配信、AI による名刺画像解析および報告書補助
Expo, Inc.米国モバイルアプリのビルド、配信、プッシュ通知トークン管理
エックスサーバー株式会社日本Web版アプリのホスティング

これらの委託先は、いずれも本ポリシーと同等以上のセキュリティ基準で個人情報を取り扱います。米国所在の委託先については、利用者が同意する範囲で個人情報を国外へ移転することがあります。

4. 利用者が入力する情報の取り扱い

本アプリにアップロードされたメッセージ、報告書、名刺画像等の内容は、原則として同じ組織内の利用者のみが閲覧可能です。当社運営者は不具合対応・不正検知等の必要が生じた場合に限り、内容にアクセスすることがあります。

5. AI（人工知能）機能による情報処理

本アプリは、利用者の利便性向上のため、Google LLC の Gemini API を利用した以下のAI処理を行います。

  • 名刺画像の文字認識および項目抽出
  • 過去の社内報告書を参照した質問応答およびドラフト生成

AI処理に送信される情報は、Google LLC のポリシーに基づき処理されます。AI処理に送信したデータをモデル学習に利用しない設定のAPIを使用しています。

6. データの保管期間

利用者情報は、利用者が当社の組織に所属する期間中保管します。退職または利用終了後は、業務上の保管義務がある期間（労働基準法等に基づく勤怠記録は5年間、その他は3年間）を経過した後、合理的な期間内に削除します。

7. 安全管理措置

  • 通信はHTTPS（TLS）で暗号化
  • 認証情報、トークンはサーバ側でハッシュ化または暗号化保管
  • データベースへのアクセスは Row Level Security（RLS）により利用者単位で制御
  • 運営者によるアクセスはログ記録
  • 外部委託先の選定および定期的なセキュリティ評価

8. 利用者の権利

利用者は、自己に関する個人情報について、以下を求めることができます。

  • 開示、内容の訂正、追加、削除
  • 利用の停止、消去、第三者提供の停止
  • 本アプリのアカウント削除

請求は本ポリシー末尾の連絡先までお問い合わせください。

9. 通知およびメッセージのブロック・通報

本アプリの利用者間で送信されたメッセージについて、利用者は他の利用者をブロックする機能、および不適切な投稿を運営者に通報する機能を利用できます。当社は通報を受け付けた場合、内容を確認し、必要に応じて当該投稿の削除、当該利用者のアクセス停止等の措置を講じます。

10. 子どもの情報

本アプリは社内利用を前提としており、13歳未満の利用は想定していません。13歳未満の利用者の情報を取得していると判明した場合、速やかに削除します。

11. 本ポリシーの変更

本ポリシーの内容は、法令の改正、サービス内容の変更等により、利用者への事前通知なく変更されることがあります。重要な変更を行う場合は、本アプリ内またはメールで通知します。
